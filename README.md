## Features

## Quick Start

```bash
git clone https://github.com/simonbiennier/nextwind
cd nextwind
yarn name:{windows|mac|linux} <project-name>
yarn
```

### Development

To start the project locally, run:

```bash
yarn dev
```

Open `http://localhost:3000` with your browser to see the result.
